# install-linux

> Softwares to install on Linux

## Usage

Just run:

```sh
bash -c "$(wget -O - https://raw.githubusercontent.com/fdaciuk/install-linux/master/install.sh)"
```

## Softwares that will be installed:

- Curl
- Vim
- Git
- Atom
- Chrome
- Docker
- KeePassX

> Enjoy it ;)
